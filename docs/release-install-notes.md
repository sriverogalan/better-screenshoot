## Get started (macOS)

Full guide: https://github.com/sriverogalan/better-screenshoot#get-started

This build is **not** signed with an Apple Developer certificate (free distribution). macOS shows a **one-time** security warning on first launch — that is normal.

1. Download the `.dmg` for your Mac (Apple Silicon `*_aarch64.dmg` or Intel `*_x64.dmg`).
2. Open the `.dmg` and drag **Better Screenshoot** to **Applications**.
3. **First launch:** Finder → **Applications** → **right-click** **Better Screenshoot** → **Open** → confirm **Open**. Do not double-click the first time.
4. Allow **Screen Recording** (and **Accessibility** for global shortcuts) when prompted.

### Instalación (español)

1. Descarga el `.dmg` de tu Mac (`*_aarch64.dmg` para Apple Silicon o `*_x64.dmg` para Intel).
2. Abre el `.dmg` y arrastra **Better Screenshoot** a **Aplicaciones**.
3. **Primera apertura:** Finder → **Aplicaciones** → **clic derecho** en **Better Screenshoot** → **Abrir** → confirmar **Abrir**. No uses doble clic la primera vez.
4. Acepta **Grabación de pantalla** (y **Accesibilidad** para atajos globales) si macOS lo pide.
